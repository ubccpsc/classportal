To install:

```npm install```

NPM run scripts have been set up for most common tasks:

* Build: ```npm run build```
* Test: ```npm run test``` (or ```npm test```)
* Test + coverage: ```npm run cover``` (HTML reports in ```./coverage/lcov-report/index.html```)
* Run: ```npm run start``` (or ```npm start```)
